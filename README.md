# FountGPT Frontend

A sleek, responsive web interface for interacting with the **FountGPT AI**, built with **React 19**, **Vite**, and **TypeScript**.

---

## 🌟 Features

- **Real-time Chat** — Seamless communication with the FountGPT AI *"Seeker"*.
- **Conversation History** — Persistent sidebar to revisit past *Searches for Truth*.
- **Dynamic Theming** — Light/Dark modes with saved user preferences.
- **Session Management** — Automatic session tracking using Local Storage + UUIDs.
- **Optimistic UI** — Instant visual feedback for a fast chat experience.

---

## 🛠️ Tech Stack

- **Framework:** React 19 — https://react.dev/
- **Build Tool:** Vite 6 — https://vitejs.dev/
- **Language:** TypeScript — https://www.typescriptlang.org/
- **Styling:** Custom CSS with variable-based theming
- **Linting:** ESLint (modern flat config)

---

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+ recommended) — https://nodejs.org/
- npm — https://www.npmjs.com/  
  or  
- yarn — https://yarnpkg.com/

---

### Installation

#### 1. Clone repository

```bash
git clone https://github.com/lucumango/fountgpt-frontend.git
cd fountgpt-frontend
```
#### 2. Install dependencies
```bash
npm install
```

#### 3. Backend configuration
The frontend expects a backend running at: 
```bash
http://localhost:8080/api/chat
```

#### 4. Run development server

```bash
npm run dev
```
App available at>
```bash
http://localhost:5173
```
